# Startup Macros
1. Go to gcode_macro.cfg
2. Go to the CLEAR_NOZZLE macro (line 57)
3. From "gcode" to wherever the next macro begins (SHAKE_OOZE) replace it with [this](https://github.com/Camden-Winder/Qidi-Q2-superuser/blob/main/Configurations/Basic%20Changes/CLEAR_NOZZLE.txt)
4. Make your way to the "PRINT_START" macro (or just control F)
5. If you have the Qidi box use [this text](https://github.com/Camden-Winder/Qidi-Q2-superuser/blob/main/Configurations/Basic%20Changes/PRINT_START%20(box).txt)

If you do not have the Qidi box, copy and paste [this text](https://github.com/Camden-Winder/Qidi-Q2-superuser/blob/main/Configurations/Basic%20Changes/PRINT-START%20(No%20Box).txt)
6. From "gcode" to the next macro "ENABLE_ALL_SENSORS" replace it with the text you coppied from the correct.txt file

# Ending Macros

Congragulations, that is it for better macros for now.
